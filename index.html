How to use this dashboard
  1. Log in with your Google account (only the admin email is allowed).
  2. Go to each tab (Logo & Resume, Projects, Certificates, Experience).
  3. Fill the form for new items or click "Edit" on existing items.
  4. Click Save. A password popup will appear.
  5. Enter the correct admin password to save changes. Data is stored in localStorage and can be sent to backend via saveToBackend().
